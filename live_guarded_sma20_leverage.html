Guarded A10/B20 SMA20 Leverage Signal

Computes the strategy state from S&P 500 daily closes: base exposure is 1x only when close is above SMA20; drawdown of 10% arms 2x and drawdown of 20% arms 3x, but recovery leverage is only allowed when price is above SMA20.

Strategy Rules

  • Base trend rule: hold 1x S&P exposure only when the close is above the 20-day SMA. Otherwise hold cash/T-bills.
  • 2x recovery tier: if the S&P is down 10% or more from its closing high-water mark, the 2x tier is armed. It can only run when price is above SMA20.
  • 3x recovery tier: if the S&P is down 20% or more from its closing high-water mark, the 3x tier is armed. It can only run when price is above SMA20.
  • Guard: if price falls back below SMA20, recovery leverage is blocked and the page falls back to the base rule.
  • Exit recovery: 2x resets after the S&P rises 25% from the tier entry close. 3x resets after the S&P rises 33.33% from the tier entry close.

The official signal uses the latest completed daily close and matches the backtest logic. The intraday signal is provisional and can use a delayed SPX quote through an API/proxy or a manually entered price.
