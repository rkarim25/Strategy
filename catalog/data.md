## Known data notes (read before downloading)

- spx_daily.csv / ndx_daily.csv cache only ~30 years (from ~1996) for the website. The FULL S&P history back to 1950-01-03 is downloaded on demand by backtest_spx_distance_scale.py and embedded in spx_distance_scale_site_data.json - it is NOT stored in spx_daily.csv. Need pre-1996 SPX? Reuse that JSON / script before re-downloading.
- Two writers touch each *_daily.csv: the cron (full precision .12g) and backtests (.9g, ~30y) - expect churn; never hand-edit them.
- gold uses GC=F (futures), which begins 2000-08; longer gold history needs a different proxy.
- msci_world is deliberately clamped to 2009-12-01 (pre-2009 Yahoo SWDA.L data is unreliable).

_All sources: Yahoo Finance via `yfinance` (`data_manager.py`). Cleaning: `price_cleaning.py`._
